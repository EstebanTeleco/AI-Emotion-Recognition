# AI Emotion Recognition 
Real-time facial emotion recognition system built with Python, OpenCV, and MediaPipe Face Mesh. 



## Features - 
Real-time face detection 
- Emotion recognition from facial landmarks 
- MediaPipe Face Mesh integration 
- OpenCV visualization 
- Lightweight and fast processing 

## Technologies 
- Python 
- OpenCV 
- MediaPipe 
- NumPy 

## Installation
bash
git clone https://github.com/EstebanTeleco/AI-Emotion-Recognition.git
cd AI-Emotion-Recognition
pip install -r requirements.txt
python -m emotion_processor.main
